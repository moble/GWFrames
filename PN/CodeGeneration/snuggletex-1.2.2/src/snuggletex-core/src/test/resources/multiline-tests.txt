# Defines multiple-line tests
#
# Input format is:
#
# LaTeX (multiple lines)
# ---- (divider)
# Resulting set of Nodes, as many lines as required.
# ==== (divider)
#
# Lines starting with '#' are documentation and are stripped out when running tests
#
##############################
Hello

There
----
<p>Hello</p>
<p>There</p>
====
Hello

There

Again
----
<p>Hello</p>
<p>There</p>
<p>Again</p>
====
Hello

\begin{itemize}
\item A
\item B
\end{itemize}

There
----
<p>Hello</p>
<ul>
  <li>A</li>
  <li>B</li>
</ul>
<p>There</p>
====
